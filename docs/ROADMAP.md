# Roadmap

## Phase 1 — MVP

Goal: core loop working end to end. No auth, no AI, no persistence.

- [ ] YouTube URL input + video embed (YouTube IFrame API)
- [ ] Subtitle fetching for public videos
- [ ] Algorithmic fill-in-the-blank generation (skip stop words, every Nth content word)
- [ ] Exercise UI synced to video playback
- [ ] Auto-pause on blank, resume on correct answer or skip
- [ ] Instant feedback (correct / incorrect)
- [ ] Basic error handling (no subtitles, invalid URL, private video)
- [ ] Deploy to production

## Phase 2 — Control

Goal: give the user control over the experience.

- [ ] Difficulty setting (how many words are hidden)
- [ ] Word category filter (nouns only, verbs only, etc.)
- [ ] Manual subtitle upload (`.srt`, `.vtt`) as fallback
- [ ] Multi-language subtitle support
- [ ] Keyboard shortcuts (skip, replay segment, hint)

## Phase 3 — Retention

Goal: give users a reason to come back.

- [ ] User accounts (email or OAuth)
- [ ] Save video history and scores
- [ ] Progress tracking per video
- [ ] Streak and summary at end of video

## Phase 4 — AI layer

Goal: make exercises smarter using AI.

- [ ] AI selects the most meaningful words to hide (not random)
- [ ] Hints and explanations generated by AI
- [ ] Open-ended questions based on video content
- [ ] Personalized word selection based on past mistakes
